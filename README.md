Reflection Summary – How Microsoft Copilot Helped Me Throughout the Project
Microsoft Copilot supported me at every stage of this project and helped me work more efficiently and confidently.

Copilot guided me in organizing my project structure by suggesting a clear and professional layout with separate folders for the front-end and back-end. It also assisted me in generating the initial integration code, including the fetch() request on the front-end and the Express route on the back-end.

During debugging, Copilot helped me identify and fix several issues such as missing middleware, CORS errors, incorrect JSON validation, and potential server crashes. It also helped me design clean and consistent JSON structures for API communication.

Copilot suggested improvements to simplify my functions, remove redundant code, and enhance performance. It encouraged best practices such as using async/await, proper error handling, and cleaner formatting.

Overall, Copilot acted as a helpful assistant throughout the project, providing explanations, corrections, and suggestions that allowed me to learn better coding habits and complete the project more efficiently
